THIRD-PARTY SOFTWARE NOTICES AND INFORMATION

CodeBrix.Develop.UI.GnomeIntrospection is a snapshot repository of GObject
Introspection (GIR) XML data files. Every .gir file in this repository is
third-party content: each is a machine-generated API description of a GNOME
platform library, produced by that library's own build process via GObject
Introspection's g-ir-scanner tool. This repository adds no source code of
its own.

================================================================================

PROVENANCE

All .gir files (31 in linux/, 25 in macos/, 26 in windows/ — the complete
set shipped by the upstream snapshot) and the three gir_files*.txt manifest
files were copied without modification, on 2026-07-05, from:

    Repository: https://github.com/gircore/gir-files
    Commit:     fa9a30ef0bedb8b8b96e87421713e6ba4698a466

That commit is the exact gir-files snapshot pinned (as the ext/gir-files git
submodule) by release tag 0.8.0 of https://github.com/gircore/gir.core
(gir.core commit 41bd0a6b0855ec4f50c2f702d394333c4c17c3f3). The gir-files
repository collects the .gir files emitted when the GNOME libraries are
built on each of the three platforms; its maintainers publish it under the
GNU Lesser General Public License, Version 2.1 — the same license reproduced
in this repository's LICENSE file.
